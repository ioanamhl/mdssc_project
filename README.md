# 🌿 GreenCart Grocery

GreenCart is a **full-stack grocery shopping web application** built using the **MERN stack (MongoDB, Express.js, React, Node.js)**.  
It allows users to browse groceries, manage their shopping cart, and purchase daily essentials through a modern and user-friendly interface.

The project focuses on building a **scalable e-commerce platform** with smooth UI, real-time cart updates, and secure user authentication.

---

## 🚀 Features

### 🛒 User Features
- User registration and login
- Browse grocery products
- Search and filter grocery items
- Add products to cart
- Remove products from cart
- Real-time cart updates
- Responsive and modern UI

### 🛠 Admin Features
- Admin dashboard
- Add new grocery products
- Update product information
- Delete products
- Manage inventory

### 🔐 Authentication
- Secure user login and registration
- JWT-based authentication
- Protected routes
